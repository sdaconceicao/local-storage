## Browser Storage 

With the evolution of frontend web applications, we've come to rely on browser storage to do more then ever before. From simple session 
information to entire databases containing files, the options available now can help you produce a complex frontend that can work 
without needing a backend or active internet connection.

### Uses

* Tracking anonymous user's navigation of a site
* Storing user config data like a theme or last page visited
* Offline data for a PWA 

### Options

* Cookies
* LocalStorage
* Indexdb
* LocalForage
* Pouchdb
